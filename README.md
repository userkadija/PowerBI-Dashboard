📊 US States Population & Census Dashboard (Power BI)

📌 Overview

This project is an interactive Power BI dashboard that analyzes and visualizes population distribution across US states and census regions. It provides insights into regional population trends, state comparisons, and historical population growth using structured census data.

The goal of this project is to demonstrate data analysis and visualization skills by transforming raw demographic data into clear, interactive, and meaningful insights.


📊 Features
Interactive map visualization of US states by census region

Breakdown of states by census regions and divisions (Midwest, South, West, Northeast, etc.)

Bar chart analysis of number of states per census region

Population trends over time across different states

Detailed data table showing state-level demographic information

Filters and slicers for dynamic data exploration


🛠 Tools & Technologies
Power BI

Data Modeling (relationships between tables)

DAX (Data Analysis Expressions)

Data Visualization techniques

US Census Population Dataset


🔍 Key Insights

The South and West regions contain more states compared to other regions

Population distribution varies significantly across states and regions

Historical data shows steady population growth across most states

Regional classification simplifies large-scale demographic analysis


🎯 Project Purpose

This project was built as part of a data analysis learning journey to strengthen skills in:

Data cleaning and modeling

Dashboard design

Insight generation from real-world datasets

Interactive reporting using Power BI
